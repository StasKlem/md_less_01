создай терминальное приложение на golang.
Приложение это интерфейс для общения с LLM.
Эндпоин для общения с LLM `https://routerai.ru/api/v1/chat/completions`

Как работает приложение:
- приложение ждет ввода пользователя
- после ввода отправляет запрос к llm
- распечатывает ввод пользователся и ответ llm
- когда пользователь вводит слово "Все" то llm должна считать что выбор закончен и 
отправить пользователю сообщение "Заказ принят"

Суть приложения:
- llm это робот для закза пиццы
- llm должна отвечать только в формате `json`
- формат 
```json
{
	"message": "сообщение от llm"
	"pizza": ["ингредиент который ввел пользователь #1", "ингредиент который ввел пользователь #2", ...]
}
```
